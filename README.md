# Temperance

A Balatro mod that shows the total sell value of all jokers directly underneath the consumables area, mirroring the information shown by the Temperance tarot card.

Compatible with [What's In My Fool](https://github.com/colonthreeing/WhatsInMyFool) — when both mods are active, the sell value appears between the Fool card name and the consumable counter (e.g. `Pluto $0 0/2`).

Temperance is based on the work done in [JokerSellValue](https://github.com/OppositeWolf770/JokerSellValue) and [What's In My Fool](https://github.com/colonthreeing/WhatsInMyFool).
