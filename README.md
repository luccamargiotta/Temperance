# Temperance

A Balatro mod that shows the total sell value of all jokers next to the jokers and/or consumables counter, mirroring the information shown by the Temperance tarot card.

![example](example.png)

## Features
- Shows total joker sell value (e.g. `$25`)
- Configurable display: choose between consumables area, jokers area, or both

## Compatibility
- [What's In My Fool](https://github.com/colonthreeing/WhatsInMyFool) — when both mods are active, the sell value appears between the Fool card name and the consumable counter (e.g. `Pluto $0 0/2`)
